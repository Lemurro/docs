# Документация Lemurro

# Требование
1. PHP v7.0.8+
2. [DAUX.io](https://dauxio.github.io)

# Установка
```bash
composer global require daux/daux.io
```

# Сборка
Необходимо находится в родительском каталоге папки `docs`
```bash
daux generate
```
