# Создание нового проекта
1. Устанавливаем копию скелета: `composer create-project lemurro/client-framework7 /my/project/path`
2. Устанавливаем зависимости: `npm install`
3. Устанавливаем библиотеки: `bower install`
4. Запускаем первичную сборку и слежку за изменениями исходников `gulp default`
5. Определяем параметры проекта: `assets/config.js`
6. Запускаем в браузере файл: `index.html`
